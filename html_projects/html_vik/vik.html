Visual Immersion Kit - vik

vik is a collection of C++ libraries developed to facilitate the implementation of proof-of-concept projects in computer vision, computer graphics, machine learning algorithms, and deep learning models. The primary goal at its inception was to design a flexible and extensible software architecture that integrates solutions from various disciplines while operating transparently across a broad range of hardware platforms. Unlike full-featured game engines, vik does not aim to offer an extensive feature set unless supported by commercial backing.
A brief overview of its constituent libraries is provided below. If you are interested, please request access by email.

vikHelper

consists of various components, including utilities, networking, and mathematics, that consolidate commonly used functionalities to support further development. Users are not required to adopt the entire library; instead, they can selectively utilize only the features relevant to their needs. The library includes certain external dependencies, and detailed licensing information for these is provided within the repository.

utility

The utility module provides a broad range of functionalities, including—but not limited to—logging, file I/O, thread pooling, and reflection.
